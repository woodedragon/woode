使用说明：
上传并解压到网站根目录，登陆后台配置网站名称/密码等基础信息即可

nginx.rewrite ：是Nginx伪静态规则，因为搜索引擎只收录首页和文章页，对列表不做静态处理

不需要伪静态的把indexs.php更名为index.php即可

网站后台：域名/Admin

默认账号：admin  密码：123456  

修改密码：登陆后台-网站配置-管理密码

程序目录结构：
Admin----后台目录
Article----文章储存路径
Category----分类储存索引
Style----样式表
Template----公共模版目录
Upload----图片上传路径
indexs.php----伪静态首页
index.php----动态首页
